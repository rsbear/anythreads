# A list of actions that this library needs to support

## Actions

**Threads**
- Create a new thread
- Delete a thread
- Edit a thread

**Replies**
- Create a reply to a thread
- Create a reply to a reply
- Delete a reply
- Edit a reply

**Votes**
- Upvote a thread
- Downvote a thread
- Upvote a reply
- Downvote a reply
